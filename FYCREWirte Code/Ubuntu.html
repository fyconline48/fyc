Home

Frist thing we need it a 32bit

sudo dpkg --add-architecture i386

Now we need to add the repo

sudo mkdir -pm755 /etc/apt/keyrings

sudo wget -O /etc/apt/keyrings/winehq-archive.key https://dl.winehq.org/wine-builds/winehq.key

Now find your distro and use the command and enter the command starting with sudo (ONLY DO ONE)

Ubuntu 22.10 | sudo wget -NP /etc/apt/sources.list.d/ https://dl.winehq.org/wine-builds/ubuntu/dists/kinetic/winehq-kinetic.sources

Ubuntu 22.04 , Linux Mint 21.x | sudo wget -NP /etc/apt/sources.list.d/ https://dl.winehq.org/wine-builds/ubuntu/dists/jammy/winehq-jammy.sources

Ubuntu 20.04 , Linux Mint 20.x | sudo wget -NP /etc/apt/sources.list.d/ https://dl.winehq.org/wine-builds/ubuntu/dists/focal/winehq-focal.sources

Ubuntu 18.04 , Linux Mint 19.x | sudo wget -NP /etc/apt/sources.list.d/ https://dl.winehq.org/wine-builds/ubuntu/dists/bionic/winehq-bionic.sources

Now we need to update our packages

sudo apt update

Now we install wine (I would say use Development branch)

Stable branch | sudo apt install --install-recommends winehq-stable

Development branch | sudo apt install --install-recommends winehq-devel

Staging branch | sudo apt install --install-recommends winehq-staging

Now when you want to run your .exe file open terminal and type cd then your file path lets say for this the downloads we would type this (cd Downloads)

And now we run the exe with wine THING.exe

Now your done
